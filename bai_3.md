# Bai 3
## Tạo một repository trên github

---
### Các bước:
#### Bước 1: truy cập github và đăng nhập/đăng ký

#### Bước 2: nhấp chuột vào dấu `+` bên phải góc trên cùng màn hình và chọn `new repository`:
<image src="image/b3_1.png">

#### Bước 3: Điền các thông tin cụ thể cho repository:
<image src="image/b3_2.png">
1. **Repository name** (Tên dự án): điền tên dự án vào đây.

2. **Description** (Mô tả dự án): đây là phần không bắt buộc

3. **Choose visibility**: Chọn chế độ hiển thị
* **Public** (Công khai): Bất kỳ ai trên Internet cũng có thể tìm thấy và xem mã nguồn của dự án này. Tuy nhiên, chỉ có bạn (hoặc những người được bạn cấp quyền) mới có thể sửa đổi và đẩy code lên.
* **Private** (Riêng tư): Chỉ có bạn và những người bạn đích danh thêm vào dự án mới có quyền xem và thao tác.

4. **Add README**: Tạo sẵn một file tên là README.md để ghi tài liệu hướng dẫn chạy dự án.Nếu định đẩy một thư mục có sẵn từ máy tính lên,không được tích chọn vào ô này

5. **Add .gitignore**: Tạo sẵn một file giúp  liệt kê những file/thư mục thừa hoặc nhạy cảm mà Git nên bỏ qua, không đẩy lên mạng (.env, node_modules ...). nếu có sẵn file này ở local thì không tích chọn

6. **Add license**: Chọn giấy phép bản quyền cho code

#### Bước 4: Enter `Create repository`
